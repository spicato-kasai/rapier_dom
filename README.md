# vite-10syunen

DOMで写真を2個たて並び、DOMはopacity:0にしてcanvasで描いてスクロールしてもDOMと同期するようにする
DOMをマウスでクリックして動かせる（置く時もクリック）

## セットアップ

```bash
npm install
```

## 開発起動

```bash
npm run dev
```

## 本番ビルド

```bash
npm run build
```

## 操作

- `リスタート` ボタンで小石の配置をリセットできます。
